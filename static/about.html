A propos


Motivation :
Le projet Web a pour vocation d'initier les étudiants de 3ème année du département Télécommunications, Services et Usages de l'INSA de Lyon à la construction d'un portail Internet, indispensable de nos jours. Nous avons donc choisi de créer un site de partage de ressources (sujets, et si possible corrigés, de TD, de TP, de DS, de projet) exclusivement accessible après inscription. L'intérêt ultime de notre site est de permettre aux utilisateurs de téléverser leurs propres documents sur le site. Architecture du site :
Afin de satisfaire le critère d'esthétisme, nous avons fait le choix d'utiliser le modèle Bootstrap en Front-end : en téléchargeant un squelette basé sur HTML ainsi qu'une structure de CSS, nous avons pu obtenir le canevas de notre site. Nous avons ensuite apporté notre touche personnelle à ceci, en ajoutant des champs tels que l'affichage d'informations personnelles, le visionnage de vidéos,.. puis en modifiant les couleurs, les dimensions, etc... Les pages ainsi créées sont figées. Afin de les dynamiser, nous incorporons du code Javascript, qui fera le lien entre ces pages et l'arrière plan où se trouve Python. D'autre part, dues aux différentes fonctionnalités offertes par notre site, il convient d'utiliser plusieurs technologies au niveau du Back-end : Flask nous permet de développer notre application en collaboration avec SQLAlchemy qui lui, permet de manipuler nos bases de données. Ces deux frameworks seront incorporés dans des fichiers Python. Ce dernier a été retenu afin de créer notre serveur car il présente la qualité de portabilité ( compatible avec Windows, MacOS et Linux). Quant à la jonction entre ces deux entités, nous optons pour JSON et Ajax. Ce dernier charge des données sans recharger toute la page, en utilisant Javascript et les requêtes sont rapides. Il vise également à satisfaire les transferts de données. Ces derniers sont exclusivement gérés par Javascript. Les données reçues sont au format JSON (il segmente les données dans un objet Javascript).
